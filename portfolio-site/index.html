Computer Vision

Computer Vision Detection App

A Flask + OpenCV DNN web app that transforms any webcam feed into a real-time object detection studio. It includes a mock inference mode, TTS guidance, and canvas overlays—perfect for demos and accessibility use cases.

  • Integrated MobileNet-SSD with graceful fallback when model assets are absent
  • Implemented 1-second frame capture loop and debounced speech synthesis
  • Designed responsive UI with detection analytics, latency, FPS, and demo assets

API & Systems

Hope Services API

An e-commerce and donations API that pairs JWT auth with SQLAlchemy, Marshmallow validation, Flamingo-ready Swagger docs, and background workers simulating Celery. Rate limiting, idempotency, and an automated test suite ensure production resilience.

  • Architected relational schema covering inventory, orders, donations, idempotency keys
  • Implemented idempotent order creation with job orchestration (receipts + inventory sync)
  • Shipped full pytest coverage, seed scripts, Docker image, and Makefile-driven workflows

Deliverables

Backend engineer · DevOps
Frontend Experience

Hope Services Frontend

A React + Vite storefront that delivers seamless shopping, donations, and admin tooling. Zustand powers resilient state, while TanStack Query modernizes API communication for a polished customer experience.

  • Crafted responsive UX for catalog, cart, checkout, donations, and admin views
  • Implemented persistent cart/auth state, order idempotency key helpers, and toast feedback
  • Integrated Tailwind for scalable theming and Vite proxying for API dev flow
